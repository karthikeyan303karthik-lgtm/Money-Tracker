# 💰 Money Tracker Pro

A professional full-stack personal finance management application built using HTML, CSS, JavaScript, Node.js, Express.js, and MySQL.

Money Tracker Pro helps users track income, expenses, budgets, and savings goals through a modern dashboard with analytics and reports.

---

## 🚀 Features

### 👤 User Authentication
- User Registration
- User Login
- JWT Authentication
- Secure Password Hashing (bcrypt)

### 💵 Transaction Management
- Add Income
- Add Expenses
- Edit Transactions
- Delete Transactions
- View Transaction History

### 📊 Dashboard Analytics
- Total Balance
- Total Income
- Total Expenses
- Monthly Summary
- Financial Overview

### 📈 Charts & Reports
- Income vs Expense Pie Chart
- Monthly Expense Analysis
- Savings Progress Visualization

### 🎯 Budget & Goals
- Monthly Budget Tracking
- Savings Goals
- Goal Progress Monitoring

### 🎨 Modern UI
- Responsive Design
- Mobile Friendly
- Professional Dashboard Layout
